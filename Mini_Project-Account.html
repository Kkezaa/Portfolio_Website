Projects

Transaction Tracker App

I built a simple transaction management web application using Flask that supports full CRUD functionality. Users can add, view, edit, delete, and search transactions based on amount ranges, as well as calculate the total balance dynamically. Through this project, I learned how to structure a Flask application, work with routes and HTTP methods, handle form data, implement basic server-side logic, and pass data to HTML templates. It also helped me better understand how backend logic connects to the front end in a real-world workflow.
